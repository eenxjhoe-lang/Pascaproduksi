⚠️ Peringatan Sistem — Perlu Perhatian
Daftar peringatan dan anomali yang terdeteksi dari seluruh pelabuhan
💡
Tentang Peringatan Sistem
Halaman ini menampilkan seluruh peringatan otomatis yang terdeteksi dari data PNBP Pascaproduksi. Peringatan dikelompokkan berdasarkan tingkat keparahan: 🔴 KRITIS (capaian < 25% atau anomali besar), 🟡 PERHATIAN (capaian 25–50% atau data tidak konsisten), 🟢 INFORMATIF (catatan tambahan). Peringatan KRITIS perlu tindak lanjut segera untuk mencapai target tahunan.
🎯 Progres Capaian PNBP
Perbandingan progres realisasi terhadap target tahunan per pelabuhan
📘
Tentang Progres Capaian PNBP
Progres Capaian PNBP adalah indikator utama kinerja pelabuhan dalam mencapai target Penerimaan Negara Bukan Pajak (PNBP) Pascaproduksi tahunan. Dihitung sebagai persentase Realisasi dibagi Target, dengan klasifikasi: KRITIS < 25%, KURANG 25–50%, SEDANG 50–75%, BAIK ≥ 75%.
💡 Insight Otomatis
Analisa otomatis yang dihasilkan dari pola data transaksi PNBP
🧠
Tentang Insight Otomatis
Insight Otomatis adalah kesimpulan cerdas yang disusun sistem berdasarkan data produksi, nilai PNBP, dan tren bulanan. Mencakup: Bulan Terbaik, Tren Produksi, Proyeksi Tahunan, Anomali Data, dan Kapal Terproduksi.

🗑 Hapus Semua Data?

Semua data dari ketiga pelabuhan akan dihapus dan dashboard akan dikosongkan. Tindakan ini tidak dapat dibatalkan.

⚙️ Hubungkan Google Sheets

📌 Cara menghubungkan:
1. Buka Google Sheets Anda → Extensions → Apps Script
2. Paste kode dari file apps-script.gs yang tersedia di paket ini
3. Klik Deploy → New deployment → Web app
4. Set "Who has access" = Anyone
5. Salin URL yang muncul, paste di bawah ini
menit
